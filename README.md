a repository for TI-84/Casio 50 (python compatible) scripts, ment to help with configuring the reactor. (MODIFIED BY ADDITION_CHEESE128!!!! only like one line lmao)

## To Run These
These scripts can be run on a TI-84 Plus CE, Casio CI-50, or a desktop computor.
Due to the variability of the installs on calculators we will only show the desktop instructions.

### Desktop Instructions:
This guide assumes that you have [Python 3.12+](https://www.python.org/downloads/)
1. Clone/Download the repository
* Click <img width="100" height="32" alt="image" src="https://github.com/user-attachments/assets/2777551c-0adf-42a8-aaa3-173af16b8609" />
* Then `Download Zip` <img width="100" height="20" alt="image" src="https://github.com/user-attachments/assets/c7e4456f-134c-4e8b-9ed1-2e08a89b1a60" />
2. Extract it
* Extract the downloaded zip file
* Ensure it is in its own folder.
3. Run It
  * Open a terminal in the folder with the extracted files
  * Run this code snippit
  `python3 ./OAKRIDGE84.py` or `python3 ./OAKRIDGE50.py`
